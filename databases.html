Databases Project

This is a simple web application to help a user work with a bank database our team created.


In this project, we used Node.js and javascript to create a front end for a SQL relational database our group created. The application allowed the user to first login, and depending on if they were an admin, manager, or customer to do different actions. For example, a manager could hire new employees, a customer could withdraw or deposit from their account, and an admin had the ability to start a new bank.

I worked with designing and creating the database in SQL. After a basic layout of the database was created, I then worked on creating the front-end and leading the group on that portion of the project.
