Pool

  • Category: Web Developement
  • Made with React, Node.js, and Firebase
  • Try it: http://simpool.tech/

Experience

This application was first created during Hackgt9. It allows for users to better organize their carpooling groups.


In this project, we created a website that allows for people to easily create carpooling groups. By going to our site, users are able to create events they would like to carpool to. Then each event has its own unique link. By sending that link to friends, they can then add themselves as drivers if they have a car or join a car as a rider. With Pool, it is easy to plan carpooling groups. Not only that, there is a built in chat function, that allows people to communicate important information for each car like pick up location and time. We also allows for people to put in information about their cars and the drivers to make pick up go smoother.

I am working as a full stack developer for this application, but mainly on the frontend side of things. Using React and the Material UI library, I have created an intuitive and appealing site that allows for ease of use on both mobile and PC.
