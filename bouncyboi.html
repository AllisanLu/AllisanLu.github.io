Bouncy Boi

C++, SFML

Bouncy Boi is a 1 player mini-game (semi-brick breaker)


This is a game where I first coded the entire code base on my own without a team. In previous games I have had the help of others and a team to help me create games. With larger teams, I have been able to create larger games that have a lot more pieces to them. With Bouncy Boi, the game is rather simple. There is just a bouncy rabbit and a player platform. The main systems in the games are collision and movement.

As I develop this game, I hope to add more physics-ish feeling with momentum so that the collision and bouncy boi reaction makes more sense when compared to real life situation.
